실험 전 특이사항

fbx 파일을 다운받았을 때, 텍스쳐파일과 fbx 파일로 나눠서 저장된다. 이때, fbx 파일 자체는 용량이 크지 않은 경우가 많다. 실험에서 사용한 나무 모델의 경우 총 262 MB중 70MB가 fbx 파일이었고, 나머지는 텍스쳐 파일이었다. 그래서 실험 조건\[100MB 이상으로 커밋을 반복]을 충족하기 위해 fbx 파일 외에 텍스쳐 파일도 커밋했다.

깃에 커밋할 때, fbx만 커밋하고 텍스쳐는 변경하지 않으면 깃은 자동으로 fbx 만 커밋하고 텍스쳐는 새로 받지 않는다. 마찬가지로, 내가 fbx, 텍스쳐 파일 중 하나만 변경하여 커밋하면 나머지 텍스쳐파일은 새로 커밋받지 않는다.



커밋 방법

: 비주얼스튜디오에 자동으로 텍스쳐파일과 fbx 파일에 1바이트씩 붙혀서 커밋하도록 코드를 짜고 터미널로 실행했다.
